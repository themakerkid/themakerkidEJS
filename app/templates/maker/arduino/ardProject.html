These are some that I have made using Arduino, ATTiny85 and Adafruit Gemma. These projects aren't fully documented yet but I will add some tutorials which will help you make these when I get time.

Knight Rider using ATTiny85 by Benjamin

This is a simple flasher made by me for my Halloween costume 2 years ago that displays the Knight Rider sequence. It was programmed using the Arduino IDE and consists of an ATTiny85 micro controller, some LEDs, a battery, a switch, vero board, a resistor and some wires.

Scrolling Message Board made by Benjamin

I made this scroller for Christmas 2015 to wish my CoderDojo Happy Christmas! It consists of an Arduino Nano and six MAX7219 controlled 8x8 Dot matrix modules. I will document this project soon, as other kids might be interested in making a scroller display as well. The project does require some soldering though. Coming soon!

Bluetooth controlled NeoPixels with Arduino (Short demo)

This is a quick demo of the Android tablet running the App which controls the lighting sequences on the nopixel LED's. A HC-05 Bluetooth module is used to receive commands from the App over bluetooth. The first part of the video shows the app connecting to the HC-05 module. The arduino receives the commands from the HC-05 over its serial interface. It then looks at the command and lights up the LED's with the chosen sequence.

Neopixel Ring with Adafruit Gemma

This project uses an Adafruit Gemma and an Adafruit Neopixel ring. The Adafruit Gemma is a tiny wearable platform board whict is powered by an Attiny85 microcontroller. It is programmable with an Arduino IDE over USB. You can use it for small projects where you don't need the full power of an Arduino.
