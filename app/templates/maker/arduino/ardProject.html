Welcome to the Projects page

These are some that I have made using Arduino, ATTiny85 and Adafruit Gemma. These projects aren't fully documented yet but I will add some tutorials which will help you make these when I get time.

Knight Rider using ATTiny85 by Benjamin

This is a simple flasher made by me for my Halloween costume 2 years ago that displays the Knight Rider sequence. It was programmed using the Arduino IDE and consists of an ATTiny85 micro controller, some LEDs, a battery, a switch, vero board, a resistor and some wires.

Message Scroller Board made by Benjamin

I made this scroller for Christmas 2015 to wish my CoderDojo Happy Christmas! It consists of an Arduino Nano and six MAX7219 controlled 8x8 Dot matrix modules. I will document this project soon, as other kids might be interested in making a scroller display as well. The project does require some soldering though. Coming soon!
