I buit this website using the Flask micro web framework.

What is Flask?

Well, Flask is a micro web framework written in Python and based on the Werkzeug toolkit and Jinja2 template engine. Jinja2 is a templating engine for Python. It generates the dynamic webpages on this website.

I used some Flask extensions (libraries) such as Flask-SQLAlchemy, Flask-Mail and Flask-WTF. Applications that use the Flask framework include Pinterest and LinkedIn. I have been coding with Flask for the last 10 months.

How can I learn Flask?

Miguel Grinberg has a Flask Mega Tutorial on his blog. He also has a Flask Web Development book. It is also quite easy to install Flask on the Raspberry Pi. There is an excellent tutorial on the Raspberry Pi website called Build a Python Web Server with Flask here.
