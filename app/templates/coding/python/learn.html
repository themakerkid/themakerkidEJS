What is Python?

An interpreted language, Python has a design philosophy which emphasizes code readability (notably using whitespace indentation to delimit code blocks rather than curly braces or keywords), and a syntax which allows programmers to express concepts in fewer lines of code than possible in languages such as C++ or Java.

How can I learn Python?

The best place to start is definitely Code Academy (Python Course). The Python course is a great introduction to both fundamental programming concepts and the Python programming language.

Tutorials Point (Python Course) is another good free learning website after you have finished Code Academy.


Or you can buy one of the books below. I would recommend Hello World before Adventures in Minecraft.

Python Basics 3/5

This is a book by Cambridge University on Python 3 only . In the end you make a drawing program where you can draw on a canvas using Tkinter.



Hello World Second Edition 5/5

This book has 26 chapters (which is a lot!) and you will first learn the basics of Python with Warren and Carter Sande and then move on to games like a virtual pet, hangman or a battle. This book covers Python 2.7 but does point out the some differences for Python 3 in some of the chapters.



Adventures in Minecraft 5/5

In this book you will control Minecraft with Python. Create an alien invasion, a complete Crafty Crossing game and a new block friend in Minecraft with Python alone. Create clocks, pyramids and a complete city (Manhattan, America to be exact). You must use Python 2.7 to try out the code because the API to connect with Minecraft only works with Python 2.7 currently.



Automate the Boring Stuff with Python 4/5

In this book you will learn ways to automate things that are very boring like if your friend is in America and you want to change the format of the dates. You can use Python to do all of this boring stuff! Python can also automatically fill in forms using Selenium.
