But what is code?

Learn about what code is by watching this cool video. You might be suprised to learn that code is all around you, in your home,in your parents car, on aeroplanes, in games consoles, in space and in your classroom on the whiteboard.

Just as people all around the world speak different languages, there are also many different coding languages. Depending on what you want to do, you will choose a particular language. These languages do different things. For example, you probably heard about Minecraft right? Well that was written in a language called Java. And this webpage that you are looking at right now uses HTML,CSS and JavaScript. If you want to control physical devices in the real world you will probably use a language called C/C++.

Join your local CoderDojo!

I have been attending Coderdojo for almost 2 years now. It's fantastic!! CoderDojo is a global movement of free, volunteer-led, community based programming clubs for young people. At a Dojo, young people, between 7 and 17, learn how to code, develop websites, apps, programs, games and explore technology in an informal and creative environment. In addition to learning to code attendees meet like minded people and are exposed to the possibilities of technology.

Learn how to create Webpages

Learn how to use HTML and CSS to make webpages. HTML is the markup language that you surround content with, to tell browsers about headings, lists, tables, etc. CSS is the stylesheet language that you style the page with, to tell browsers to change the color, font, layout, and more.
