But what is code?

So how do I get started?

If you want to get into coding, here are some suggestions. Just hover and click on any of the sections below:

I would recommend joining your local CoderDojo. If you cannot find a Dojo near you, you can still get started by using free resources on the Internet. The sections below have links to learning sites. Now what language should you choose? Well that really depends on your experience level, por what thpe of Apps you wish to develop. If you have no coding experience, and you are 10 or under Scratch is a good start. It uses visual coding block to build cool animations, stories and games.

But maybe you want to learn how to develop your own website? In that case you will need to start by learning HTML and CSS first, followed by JavaScript.

If you want to develop Mobile Apps, MIT App Inventor is a good first choice for Android devices. It uses code blocks just like Scratch. Then when you want to develop more advanced Apps for Android, you can move onto Java.

If you want to start on a good general programming language, then Python is an excellent choice. You can use Python for creating Games, controlling physical objects with tyhe Raspberry PI, and building the server side of Websites and doing cool maths and Science stuff.

Learn how to code Mobile Apps.

App Inventor for Android is an open-source web application originally provided by Google, and now maintained by the MIT. It allows newcomers to computer programming to create software applications for the Android OS. It uses a graphical interface, very similar to Scratch and the StarLogo TNG user interface, which allows users to drag-and-drop visual objects to create an application that can run on Android devices.

Learn how to create Webpages

Learn how to use HTML and CSS to make webpages. HTML is the markup language that you surround content with, to tell browsers about headings, lists, tables, etc. CSS is the stylesheet language that you style the page with, to tell browsers to change the color, font, layout, and more.


Learn the Processing JS language on Khan Academy

Processing JS is a JavaScript port of Processing, a programming language designed to write visualizations, images, and interactive content. It allows web browsers to display animations, visual applications, games and other graphical rich content without the need for a Java applet or Flash plugin.

Learn the Python Language.

An interpreted language, Python has a design philosophy which emphasizes code readability (notably using whitespace indentation to delimit code blocks rather than curly braces or keywords), and a syntax which allows programmers to express concepts in fewer lines of code than possible in languages such as C++ or Java.
