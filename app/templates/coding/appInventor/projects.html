How to use .apk files (You must read this to move on)

All the apps on this page will be downloaded via .apk. You will have to have an Android device to use .apk files.

Just don't forget to go to Settings > Security and check Unknown Sources.

If you're visiting this website via a computer, follow these steps:

  1. Download the .apk file on to your computer. On this website you will just hit the Green Buttons.
  2. Plug in your Android device via USB Cable. You might be asked if you want to charge your phone or to connect it as a media device.
  3. Choose Media Device.
  4. Copy the .apk file on to your phone (it can be any folder on the phone).
  5. Find the .apk file and right-click on it and hit Install.
  6. That's it!

If you're visiting this website via a tablet or a phone, you will follow these steps.

  1. Just hit the appropiate Green Button.
  2. Go to your Downloads.
  3. Tap on the .apk file.
  4. Tap Yes when prompted.
  5. That's it!
