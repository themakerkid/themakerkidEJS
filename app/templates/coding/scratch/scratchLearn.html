Some Books that you may find useful

The Scratch Wiki page is an excellent Scratch Wiki place to start. Also there are excellent projects and tutorials on the CoderDojo Kata page.

Super Scratch Programming Adventure 5/5

This is about a cat called Scratchy, a guy called Mitch and some cosmic defenders. You will learn to program in a simple language called Scratch and your projects will be part of the adventure. You will defeat the Dark Wizard in this adventure. There is a book for Scratch 1.4 and a book for Scratch 2.0. I believe this would be a good book for young kids because it has cartoon characters.



Scratch Programming. In easy steps. 4/5

This is a book that covers both versions of Scratch which is pretty cool . You will make some games and have some fun! You can do this on a Raspberry Pi as well!



Scratch 2.0 Game Development 4/5

In this book you will create games that already exist but in Scratch. Games like Mario and a Tower game (as you can see on the front cover of the book). Very good book but it is unfortunately not in colour. It only covers Scratch 2.0 (uses some new features such as the More Blocks palette).



Learn to Program with Scratch 4/5

In this advanced book (that is why it is last) you will create some mini games like a quadrilateral (four-sided shape) identifier. Covers Scratch 2.0 only. Beware! Don't be worried if you buy this book and you do not understand some of the formulae, (I don't understand some of them as well!) what matters is you understand what the code means.
